# Anteiku - Social & Guild Management App

Anteiku is a comprehensive React Native application built with Expo. It serves as a social platform where users can communicate, create and join guilds, share posts, and manage their personalized profiles.


* **Frontend:** React Native, Expo, React Navigation
* **Backend/BaaS:** Firebase (Authentication, Firestore)
* **Media Storage:** Cloudinary
* **State Management:** React Context API
* **External APIs:** RAWG (Games), TMDB (Movies/Series), iTunes (Music), Giphy (GIFs)

* **Global Feed:** Share posts with text, images, and repost functionality.
* **Guild System:** Create, join, and manage guilds. Real-time guild chats and member management.
* **Direct Messaging:** Real-time 1-on-1 chats with voice recording, image/file sharing, and emoji/GIF support.
* **Rich Profiles:** Customizable profiles with avatars, banners, statuses, and "Favorites" (Games, Music, Anime/Movies) powered by external APIs.
* **Security:** Environment variables for all sensitive API keys.
